5️⃣ 공통 필수 단축키 (외워야 함)
기능	단축키
파일 열기	Cmd + P
전체 검색	Cmd + Shift + F
명령 팔레트	Cmd + Shift + P
주석 토글	Cmd + /
실행 취소	Cmd + Z
🎯 바이브코딩 추천 손 위치
왼손: Cmd / Shift / Option
오른손: K / L / P / Enter
마지막 요약

단축키는 속도가 아니라 사고 흐름을 끊지 않기 위한 도구다. Cmd + K (AI) / Cmd + P (이동) 이 두 개만 몸에 익혀라.
