GUI vs CLI
GUI (Graphic User Interface)
마우스로 클릭클릭
폴더 아이콘 더블클릭으로 열기
드래그앤드롭으로 파일 이동
CLI (Command Line Interface)
글자로 명령 입력
cd 폴더명
클릭하여 복사
 으로 이동
mv 파일명 목적지
클릭하여 복사
 로 이동
비유로 이해하기
GUI = 식당에서 메뉴판 보고 손가락으로 가리키며 주문
CLI = 주방에 직접 들어가서 말로 주문

pwd

복사

결과: /Users/내이름/Documents

ls

복사

복사

mkdir 새폴더

복사

clear

자주 하는 실수와 해결
실수 1: "No such file or directory"
원인: 폴더/파일 이름을 잘못 입력
해결: ls로 정확한 이름 확인 후 다시 입력

클릭하여 복사
복사
실수 2: "Permission denied"
원인: 권한 없는 폴더에 접근
해결: 다른 폴더로 이동하거나 sudo 사용 (위험!)

클릭하여 복사
복사

Windows vs Mac 차이
동작	Mac/Linux	Windows (cmd)
목록 보기	ls	dir
현재 위치	pwd	cd (인자 없이)
