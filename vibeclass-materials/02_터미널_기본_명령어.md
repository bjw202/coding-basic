윈도우 · 맥북 터미널 기본 명령어 치트시트

개발·서버·자동화 작업에서 바로 쓰는 핵심 명령어만 정리

1. 현재 위치 / 이동
pwd            # 현재 경로 (맥/리눅스)
cd             # 홈 이동
cd 폴더명
cd ..          # 상위 폴더
cd ~           # 홈 디렉토리

복사

윈도우(cmd)

cd

복사
2. 목록 보기
ls             # 목록
ls -l          # 상세
ls -a          # 숨김 포함

복사

윈도우(cmd)

dir

복사
3. 폴더 생성 / 삭제
mkdir test
rmdir test             # 비어 있을 때
rm -rf test            # ⚠️ 강제 삭제 (맥/리눅스)

복사

윈도우(cmd)

mkdir test
rmdir /s /q test

복사
4. 파일 생성 / 복사 / 이동 / 삭제
touch a.txt            # 파일 생성 (맥)
cp a.txt b.txt         # 복사 (맥)
mv a.txt b.txt         # 이동/이름변경 (맥)
rm a.txt               # 삭제 (맥)

복사

윈도우(cmd)

type nul > a.txt
copy a.txt b.txt
move a.txt b.txt
del a.txt

복사
5. 파일 내용 보기
cat a.txt              # 전체 출력
less a.txt             # 스크롤(q 종료)
head a.txt             # 앞부분
tail a.txt             # 뒷부분
tail -f log.txt        # 실시간 로그

복사

PowerShell

Get-Content a.txt

복사
6. 검색
find . -name "*.js"   # 파일 찾기 (맥)
grep "error" log.txt  # 내용 검색 (맥)

복사

윈도우(cmd)

dir /s *.js
findstr error log.txt

복사
7. 프로세스
ps aux                 # 프로세스 목록 (맥)
kill -9 PID            # 종료 (맥)

복사

윈도우(cmd)

tasklist
taskkill /PID 1234 /F

복사
8. 네트워크
ping google.com
curl https://example.com

복사
9. 환경 변수
env                    # 전체 확인 (맥)
export NODE_ENV=prod   # 임시 설정 (맥)

복사

윈도우(cmd)

set
set NODE_ENV=prod

복사
10. 개발자 필수
node -v
npm -v
python --version

복사

가상환경 종료

deactivate
source deactivate      # pyenv-virtualenv

복사

rm -rf /
rm -rf *
