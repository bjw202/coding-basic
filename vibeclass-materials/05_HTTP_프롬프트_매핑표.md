네트워크·HTTP 용어 → Claude 프롬프트 매핑표

목적: 용어를 알면 곧바로 ‘안전한 지시문’으로 바꿀 수 있게 만드는 실전 표 사용법: 문제 상황의 용어를 찾고 → 그대로 프롬프트에 복붙

1️⃣ HTTP 메서드
용어	의미	Claude에게 이렇게 지시
GET	조회	GET 메서드는 유지하고, 응답 스키마만 검증해줘
클릭하여 복사

POST	생성	POST /resource 생성 시 201을 반환하고, 중복은 409로 처리해줘
클릭하여 복사

PUT	전체 수정	PUT은 멱등성을 보장하도록 구현해줘
클릭하여 복사

PATCH	부분 수정	PATCH로 부분 수정하고 기존 필드는 유지해줘
클릭하여 복사

DELETE	삭제	DELETE 요청은 멱등하게 처리하고 204로 응답해줘
클릭하여 복사
2️⃣ HTTP 상태 코드
상태코드	상황	Claude 프롬프트 예시
200	정상	성공 시 200으로 통일하고 응답 구조는 유지해줘
클릭하여 복사

201	생성	리소스 생성 성공 시 201을 반환해줘
클릭하여 복사

204	응답 없음	삭제 성공 시 바디 없이 204로 응답해줘
클릭하여 복사

400	검증 실패	입력값 검증 실패는 400으로 처리해줘
클릭하여 복사

401	인증 실패	인증되지 않은 요청은 401로 처리하고 메시지는 통일해줘
클릭하여 복사

403	권한 없음	권한 없는 접근은 403으로 명확히 분리해줘
클릭하여 복사

404	없음	존재하지 않는 리소스는 404로 응답해줘
클릭하여 복사

409	충돌	중복 생성은 409로 처리해줘
클릭하여 복사

500	서버 오류	예상치 못한 오류만 500으로 보내고 로그를 남겨줘
클릭하여 복사

504	타임아웃	외부 호출 타임아웃 시 504로 매핑해줘
클릭하여 복사
3️⃣ API 계약 / 스키마
용어	의미	Claude 프롬프트
API 계약	요청/응답 약속	기존 API 계약은 절대 변경하지 말고 내부 구현만 수정해줘
클릭하여 복사

응답 스키마	JSON 구조	응답 필드 이름과 구조는 그대로 유지해줘
클릭하여 복사

버전업	변경 관리	응답 변경이 필요하면 v2 API로 분리해줘
클릭하여 복사
4️⃣ 인증 / 인가
용어	의미	Claude 프롬프트
인증	신원 확인	JWT 인증 로직은 유지하고 메시지만 수정해줘
클릭하여 복사

인가	권한 확인	권한 체크 로직을 추가하되 기존 인증 흐름은 유지해줘
클릭하여 복사

토큰 만료	세션 종료	토큰 만료 시 401로 응답하고 재로그인 유도해줘
클릭하여 복사
5️⃣ CORS / 브라우저 이슈
용어	의미	Claude 프롬프트
CORS	도메인 제한	CORS 설정만 수정하고 API 로직은 건드리지 말아줘
클릭하여 복사

Preflight	사전 요청	OPTIONS 프리플라이트 요청을 허용해줘
클릭하여 복사
6️⃣ 캐시 / 성능
용어	의미	Claude 프롬프트
캐시	응답 저장	GET 응답에 Cache-Control 헤더를 추가해줘
클릭하여 복사

no-store	저장 금지	민감 정보 응답에는 no-store를 설정해줘
클릭하여 복사

max-age	유효시간	캐시 유효 시간을 60초로 설정해줘
클릭하여 복사
7️⃣ 타임아웃 / 재시도
용어	의미	Claude 프롬프트
Timeout	응답 지연	외부 API 호출에 3초 타임아웃을 설정해줘
클릭하여 복사

Retry	재시도	멱등 요청에만 재시도를 적용해줘
클릭하여 복사
8️⃣ 로깅 / 운영
용어	의미	Claude 프롬프트
로그	기록	에러 발생 시 requestId와 함께 로그를 남겨줘
클릭하여 복사

모니터링	관측	장애 추적을 위해 핵심 지표를 로그로 남겨줘
클릭하여 복사
9️⃣ 장애 상황 → 즉시 프롬프트
증상	바로 쓰는 프롬프트
프론트 흰 화면	API 응답 스키마 변경 여부부터 점검해줘
클릭하여 복사

간헐적 500	null/undefined 경로를 전부 방어해줘
클릭하여 복사

트래픽 폭주	동시 요청 증가 시 병목 지점을 찾아줘
클릭하여 복사

로그인 실패	401/403을 명확히 분리해서 처리해줘
클릭하여 복사
