1️⃣ Cursor (AI 코드 에디터)
다운로드
공식 사이트: https://cursor.sh
macOS / Windows 공통 제공
설치
macOS: .dmg
클릭하여 복사
 다운로드 → Applications로 드래그
Windows: .exe
클릭하여 복사
 실행 → 설치
최초 실행 & 설정
Cursor 실행
GitHub 또는 Google 계정 로그인
Settings → AI Model 확인
기본 단축키 익히기
Cmd + K
클릭하여 복사
 : AI 명령
Cmd + L
클릭하여 복사
 : 코드 설명 요청
바이브코딩 기본 세팅
프로젝트 열기: File → Open Folder
클릭하여 복사
Git 연동 자동 활성화
2️⃣ Zed (초고속 에디터, Rust 기반)
다운로드
https://zed.dev
설치 (macOS)
brew install --cask zed

복사
Windows / Linux
현재는 macOS 중심, 프리뷰 제공
기초 설정
Settings(JSON)에서 테마/폰트 설정
Git 자동 연동
특징
매우 빠름
AI 기능은 최소 → 순수 코딩 집중용
