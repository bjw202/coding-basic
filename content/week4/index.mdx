# 4주차: Docker/배포/서버/보안/CI/CD/도메인

마지막 주에는 만든 애플리케이션을 실제 서버에 배포하는 방법을 배웁니다.
Docker, AWS, CI/CD, 도메인 설정까지 운영 환경의 핵심을 다룹니다.

## 학습 목표

- Docker로 애플리케이션을 컨테이너화할 수 있다
- AWS EC2에 서버를 배포하고 Nginx로 리버스 프록시를 설정할 수 있다
- GitHub Actions로 CI/CD 파이프라인을 구성할 수 있다
- 도메인을 연결하고 HTTPS를 적용할 수 있다

## 회차별 내용

### [10회차: Docker](/week4/session10)

Dockerfile 작성, 멀티스테이지 빌드, Docker Compose, 로컬 환경 재현

### [11회차: AWS EC2 + Nginx](/week4/session11)

EC2 인스턴스 생성, SSH 접속, Nginx 리버스 프록시, HTTPS 설정, 보안 기본

### [12회차: CI/CD + 도메인](/week4/session12)

GitHub Actions CI/CD, Vercel/GCP 배포, Cloudflare DNS, SSL/TLS 설정
