# 1주차: 코딩/웹 개발 기초 + 개발환경 세팅

첫 주에는 웹 개발의 기초를 다집니다. 개발 환경을 설정하고, 프론트엔드와 백엔드의 개념을 이해하며,
웹에서 데이터가 어떻게 오가는지 배웁니다.

## 학습 목표

- 개발 환경을 구성하고 Git으로 코드를 관리할 수 있다
- 프론트엔드와 백엔드의 역할을 이해하고 Node.js로 간단한 서버를 만들 수 있다
- HTTP 메서드와 상태 코드를 이해하고 REST API를 호출할 수 있다
- CORS 에러의 원인을 파악하고 해결할 수 있다

## 회차별 내용

### [1회차: 개발 환경 설정](/week1/session1)

Cursor/VSCode 세팅, 터미널 사용법, 프로젝트 구조 이해, Git 기초

### [2회차: 프론트엔드/백엔드 + Node.js 기초](/week1/session2)

FE/BE 역할 구분, Node.js 런타임, 모듈 시스템, Express로 API 서버 구축

### [3회차: HTTP/REST, JSON, CORS](/week1/session3)

HTTP 프로토콜, REST 설계, JSON 데이터 형식, CORS 이해와 해결
