# 3주차: DB/인증/AI API 연동

세 번째 주에는 데이터를 저장하고, 사용자를 인증하고, AI API를 연동하는 방법을 배웁니다.
풀스택 애플리케이션의 핵심 기능을 다룹니다.

## 학습 목표

- 관계형 데이터베이스(PostgreSQL)로 CRUD를 구현할 수 있다
- 문서형 데이터베이스(MongoDB)와 검색엔진(ElasticSearch)의 차이를 이해한다
- OAuth 흐름을 이해하고 소셜 로그인을 구현할 수 있다
- AI API를 서버에서 안전하게 호출할 수 있다
