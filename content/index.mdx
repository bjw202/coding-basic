# 바이브 코딩 1개월 부트캠프

1개월 만에 풀스택 웹 개발의 핵심을 배우는 집중 부트캠프에 오신 것을 환영합니다.

## 부트캠프 개요

이 부트캠프는 웹 개발의 전체 흐름을 **4주, 총 12회차** 수업으로 압축하여 전달합니다.
프론트엔드부터 백엔드, 데이터베이스, 인증, 배포까지 실무에서 꼭 필요한 기술을 다룹니다.

## 주차별 커리큘럼

### [1주차: 웹 개발 기초](/week1)

- 개발 환경 설정 (Cursor, VSCode, Git)
- 프론트엔드/백엔드 개념과 Node.js 기초
- HTTP/REST, JSON, CORS 핵심

### [2주차: React와 Next.js](/week2)

- React 기초 (컴포넌트, 상태, 이벤트)
- Next.js App Router와 서버 컴포넌트
- 비동기 프로그래밍과 디버깅

### [3주차: 데이터베이스, 인증, AI](/week3)

- PostgreSQL과 Supabase
- MongoDB와 ElasticSearch 개요
- OAuth 인증과 AI API 연동

### [4주차: DevOps와 배포](/week4)

- Docker 컨테이너화
- AWS EC2 + Nginx 배포
- CI/CD 파이프라인과 도메인 설정

## 수업 방식

각 회차는 **이론 + 실습** 구조로 진행됩니다. 매 수업마다 직접 코드를 작성하고,
실제 동작하는 결과물을 만들어 봅니다.

## 사전 준비

- 기본적인 컴퓨터 사용 능력
- 코딩에 대한 호기심과 열정
- 노트북 (macOS 또는 Windows)
