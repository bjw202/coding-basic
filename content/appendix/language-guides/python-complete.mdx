## 4. 문자열 처리

```python
# f-string (Python 3.6+, 권장)
name = "홍길동"
age = 25
print(f"이름: {name}, 나이: {age}")

# 문자열 메서드
text = "Hello, Python!"
print(text.upper())          # "HELLO, PYTHON!"
print(text.lower())          # "hello, python!"
print(text.strip())          # 앞뒤 공백 제거
print(text.replace("Python", "World"))  # "Hello, World!"
print(text.split(", "))      # ["Hello", "Python!"]
print(", ".join(["a", "b"])) # "a, b"
```

## 5. 리스트 (List)

```python
fruits = ["사과", "바나나", "오렌지"]

# 기본 조작
fruits.append("포도")      # 끝에 추가
fruits.insert(1, "키위")   # 특정 위치에 삽입
fruits.remove("바나나")    # 값으로 제거
fruits.pop()               # 마지막 제거
fruits.sort()              # 정렬

# 슬라이싱
print(fruits[1:3])         # 인덱스 1~2
print(fruits[::-1])        # 뒤집기

# 리스트 컴프리헨션 (Python의 강력한 기능)
numbers = [1, 2, 3, 4, 5]
squared = [n**2 for n in numbers]           # [1, 4, 9, 16, 25]
evens = [n for n in numbers if n % 2 == 0]  # [2, 4]
```

## 6. 딕셔너리 (Dictionary)

```python
user = {
    "name": "홍길동",
    "age": 30,
    "email": "hong@example.com"
}

# 접근
print(user["name"])           # "홍길동"
print(user.get("phone", "없음"))  # "없음" (키 없을 때 기본값)

# 수정
user["age"] = 31
user["phone"] = "010-1234-5678"

# 딕셔너리 컴프리헨션
squares = {n: n**2 for n in range(1, 6)}
# {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}
```

## 7. 함수

```python
# 기본 함수
def greet(name, greeting="안녕하세요"):
    return f"{greeting}, {name}!"

print(greet("홍길동"))           # "안녕하세요, 홍길동!"
print(greet("김철수", "반가워요")) # "반가워요, 김철수!"

# 가변 인자
def sum_all(*args):
    return sum(args)

sum_all(1, 2, 3, 4, 5)  # 15

# 키워드 가변 인자
def create_user(**kwargs):
    return kwargs

create_user(name="홍길동", age=30)

# 람다 함수
double = lambda x: x * 2
double(5)  # 10
```

## 8. 클래스와 객체지향

```python
class User:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        return f"안녕하세요, {self.name}입니다 (나이: {self.age})"

    def __repr__(self):
        return f"User(name={self.name}, age={self.age})"

# 상속
class Admin(User):
    def __init__(self, name, age, permissions):
        super().__init__(name, age)
        self.permissions = permissions

    def greet(self):
        return f"[관리자] {super().greet()}"

user = User("홍길동", 30)
admin = Admin("김관리", 35, ["read", "write", "delete"])
```

## 9. 예외 처리

```python
try:
    number = int(input("숫자 입력: "))
    result = 100 / number
    print(f"결과: {result}")
except ValueError:
    print("숫자를 입력해주세요!")
except ZeroDivisionError:
    print("0으로 나눌 수 없습니다!")
except Exception as e:
    print(f"예상치 못한 오류: {e}")
finally:
    print("항상 실행됩니다")
```

## 핵심 포인트 정리

- Python은 들여쓰기(Indentation)가 코드 구조를 결정합니다. 일관된 들여쓰기가 필수입니다
- 리스트 컴프리헨션은 Python의 강력한 기능입니다. 반복문을 한 줄로 표현할 수 있습니다
- `with` 문을 사용하면 파일, DB 연결 등의 자원을 안전하게 관리할 수 있습니다
- JavaScript의 `null` = Python의 `None`, `console.log` = `print`, `let/const` = 그냥 변수명 입니다
- 가상환경(venv)을 사용하면 프로젝트마다 다른 패키지 버전을 격리할 수 있습니다
