## Part 1: Claude Code

### Claude Code란?

Anthropic의 공식 에이전틱 코딩 도구로, 터미널에서 실행되며 코드베이스를 이해하고 자연어 명령으로 코딩 작업을 수행합니다.

**핵심 기능:**
- 루틴 작업 자동 실행
- 복잡한 코드 설명
- Git 워크플로우 처리
- 코드베이스 전체 이해
- 터미널, IDE, GitHub 통합

**요금제:**
- Pro ($20/월): 중간~높은 워크로드
- Max5 ($100/월): 5배 토큰, 집중 작업용
- Max20 ($200/월): 20배 토큰, 자율 개발용

### 설치

```bash
# macOS/Linux (권장)
curl -fsSL https://claude.ai/install.sh | bash

# Homebrew (macOS/Linux)
brew install --cask claude-code

# Windows (권장)
irm https://claude.ai/install.ps1 | iex

# 버전 확인
claude --version
```

### CLI 명령어

| 명령어 | 설명 | 예시 |
|--------|------|------|
| `claude` | 대화형 REPL 시작 | `claude` |
| `claude "query"` | 초기 프롬프트와 함께 시작 | `claude "이 프로젝트 설명해줘"` |
| `claude -p "query"` | SDK로 쿼리 후 종료 | `claude -p "이 함수 설명해줘"` |
| `cat file \| claude -p` | 파이프로 콘텐츠 처리 | `cat logs.txt \| claude -p "분석해줘"` |
| `claude -c` | 최근 대화 계속 | `claude -c` |

### 슬래시 명령어 (대화 중)

| 명령어 | 기능 |
|--------|------|
| `/help` | 사용 가능한 명령어 목록 |
| `/clear` | 대화 내역 초기화 |
| `/compact` | 대화 압축 (토큰 절약) |
| `/model` | 사용 모델 확인/변경 |
| `/cost` | 토큰 사용량 확인 |

## Part 2: OpenAI Codex CLI

### Codex CLI란?

OpenAI의 공식 CLI 코딩 에이전트. GPT-4o 기반으로 터미널에서 작동합니다.

```bash
# 설치
npm install -g @openai/codex

# 실행
codex

# API 키 설정
export OPENAI_API_KEY=sk-xxxx
```

### Claude Code vs Codex 비교

| 항목 | Claude Code | Codex CLI |
|------|------------|-----------|
| 모델 | Claude 3.5/3.7 | GPT-4o |
| 강점 | 장문 컨텍스트, 코드 이해 | 빠른 응답, 코드 생성 |
| 요금 | Pro $20/월부터 | API 종량제 |
| 파일 접근 | 전체 레포 | 현재 디렉토리 |

## Part 3: 효율적인 프롬프트 작성

### 황금 규칙

```
[범위] 어디를 수정할지 명시
[목적] 무엇을 달성할지 명시
[제약] 건드리면 안 되는 것 명시
[검증] 어떻게 확인할지 명시
```

### 잘 작동하는 패턴

```
# 구체적인 지시
src/api/users.ts 파일에서 POST /users API에 이메일 중복 검사를 추가해줘
기존 응답 스키마는 유지하고, 중복이면 409를 반환해줘

# 에러 기반 디버깅
아래 에러가 발생해. 원인을 분석하고 수정해줘:
TypeError: Cannot read property 'id' of undefined at line 87
```

### 피해야 할 패턴

```
# 너무 광범위한 지시 (금지)
전체 코드를 더 좋게 만들어줘
알아서 최적화해줘
전반적으로 리팩토링해줘
```

## Part 4: CLAUDE.md 설정

프로젝트 루트에 `CLAUDE.md` 파일을 만들면 Claude Code가 자동으로 읽습니다.

```markdown
# 프로젝트 규칙

## 수정 범위
- src/ 폴더만 수정
- node_modules, dist 무시

## API 계약
- 기존 API 응답 스키마 변경 금지
- 변경 시 v2로 버전업

## 코드 스타일
- TypeScript 사용
- ESLint + Prettier 설정 준수
```

## Part 5: MCP (Model Context Protocol) 서버

MCP는 Claude Code에 외부 도구를 연결하는 표준 프로토콜입니다.

### 유용한 MCP 서버

| 서버 | 기능 |
|------|------|
| filesystem | 파일 시스템 직접 접근 |
| github | GitHub API 연동 |
| postgres | PostgreSQL 직접 쿼리 |
| brave-search | 웹 검색 |

### 설정 방법

```json
// .mcp.json
{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/path/to/project"]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "ghp_..."
      }
    }
  }
}
```

## 핵심 포인트 정리

- Claude Code는 레포 전체 이해가 강점입니다. 대규모 리팩토링과 멀티파일 작업에 특히 유용합니다
- 프롬프트에 범위(어디), 목적(무엇), 제약(금지)을 명시하면 사고가 급감합니다
- CLAUDE.md는 팀의 AI 규칙서입니다. 프로젝트 시작 시 반드시 만드세요
- MCP 서버를 통해 데이터베이스, GitHub 등 외부 도구와 직접 연동할 수 있습니다
