# 언어 가이드 개요

수업에서 배운 내용을 더 깊이 이해하고 싶을 때 참고하는 심화 언어 가이드 모음입니다. 각 가이드는 기초 개념부터 실전 패턴까지 체계적으로 정리되어 있습니다.

## 어떤 세션 다음에 읽을까?

| 세션 | 추천 자료 | 이유 |
|------|----------|------|
| Session 2 이후 | [JavaScript 완전 가이드](/appendix/language-guides/javascript-complete) | Node.js/Express 이후 JS 심화가 필요합니다 |
| Session 4 이후 | [JavaScript 완전 가이드](/appendix/language-guides/javascript-complete) | React를 이해하려면 JS 기반이 탄탄해야 합니다 |
| Session 1 이후 | [Git & GitHub Actions](/appendix/language-guides/git-github-actions) | 개발환경 다음엔 Git 심화를 익히세요 |
| 전 과정 | [AI 코딩 가이드](/appendix/language-guides/ai-coding-guide) | 모든 세션에서 AI를 활용할 수 있습니다 |

## 자료 목록

### 언어

- **[JavaScript 완전 가이드](/appendix/language-guides/javascript-complete)** — 변수, 자료형, 함수, 객체, 비동기 프로그래밍, 모던 ES6+ 문법까지. React와 Node.js를 이해하기 위한 JavaScript 기반 가이드입니다.
- **[Python 완전 가이드](/appendix/language-guides/python-complete)** — Python 기초 문법부터 함수, 클래스, 파일 처리, 라이브러리 활용까지. Django/FastAPI 백엔드나 데이터 분석으로 확장할 수 있는 기반을 다집니다.

### 도구와 워크플로우

- **[Git & GitHub Actions](/appendix/language-guides/git-github-actions)** — Git 기본 개념과 필수 명령어, GitHub 협업 워크플로우, GitHub Actions로 CI/CD 파이프라인 구축까지 다룹니다.
- **[AI 코딩 가이드](/appendix/language-guides/ai-coding-guide)** — Claude Code와 OpenAI Codex CLI 사용법, 효율적인 프롬프트 작성, Skills와 MCP 설정 방법을 다룹니다.
- **[브라우저 개발자 도구](/appendix/language-guides/browser-devtools)** — Chrome DevTools의 Console, Elements, Sources, Network, Application 패널을 활용한 실전 디버깅 기법입니다.

## 핵심 포인트 정리

- JavaScript를 깊이 이해할수록 React, Node.js, TypeScript 학습이 빨라집니다
- Git은 처음엔 외워서 쓰고, 나중엔 원리를 이해하게 됩니다. 이 가이드는 그 두 단계를 모두 다룹니다
- 브라우저 개발자 도구를 자유자재로 쓸 수 있으면 디버깅 시간이 절반으로 줄어듭니다
- AI 코딩 가이드는 모든 세션에서 공통으로 적용할 수 있습니다
