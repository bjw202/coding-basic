## 4. 문자열

```python

text = "Hello, Python!"

print(text.strip())              # 앞뒤 공백 제거

print(", ".join(["a", "b", "c"])) # "a, b, c"
print(len(text))                 # 15
print("Python" in text)          # True
```

## 6. 반복문

```python
# for 반복문
fruits = ["사과", "바나나", "오렌지"]
for fruit in fruits:
    print(fruit)

# range
for i in range(5):       # 0, 1, 2, 3, 4
    print(i)

for i in range(1, 6):    # 1, 2, 3, 4, 5
    print(i)

# enumerate (인덱스와 값 동시에)
for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")

# while 반복문
count = 0
while count < 5:
    print(count)
    count += 1

# break, continue
for i in range(10):
    if i == 3:
        continue   # 3은 건너뜀
    if i == 7:
        break      # 7에서 종료
    print(i)
```

## 7. 리스트 (List)

```python
fruits = ["사과", "바나나", "오렌지"]

# 기본 조작
fruits.append("포도")      # 끝에 추가
fruits.insert(1, "키위")   # 특정 위치에 삽입
fruits.remove("바나나")    # 값으로 제거
popped = fruits.pop()      # 마지막 제거 및 반환
fruits.sort()              # 정렬
fruits.reverse()           # 뒤집기

# 슬라이싱
numbers = [0, 1, 2, 3, 4, 5]
print(numbers[1:4])    # [1, 2, 3]
print(numbers[:3])     # [0, 1, 2]
print(numbers[::2])    # [0, 2, 4] (2씩 건너뜀)
print(numbers[::-1])   # [5, 4, 3, 2, 1, 0] (뒤집기)

# 리스트 컴프리헨션 (Python의 강력한 기능)
squares = [n**2 for n in range(1, 6)]  # [1, 4, 9, 16, 25]
evens = [n for n in range(10) if n % 2 == 0]  # [0, 2, 4, 6, 8]
```

## 8. 딕셔너리 (Dictionary)

```python
user = {
    "name": "홍길동",
    "age": 30,
    "email": "hong@example.com"
}

# 접근
print(user["name"])                   # "홍길동"
print(user.get("phone", "없음"))     # "없음" (키 없을 때 기본값)

# 수정
user["age"] = 31
user["phone"] = "010-1234-5678"

# 반복
for key, value in user.items():
    print(f"{key}: {value}")

# 딕셔너리 컴프리헨션
squares = {n: n**2 for n in range(1, 6)}
# {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}
```

## 9. 함수

```python
# 기본 함수 정의
def greet(name, greeting="안녕하세요"):
    return f"{greeting}, {name}!"

print(greet("홍길동"))           # "안녕하세요, 홍길동!"
print(greet("김철수", "반가워요")) # "반가워요, 김철수!"

# 가변 인자
def sum_all(*args):
    return sum(args)

sum_all(1, 2, 3, 4, 5)  # 15

# 키워드 인자
def create_user(**kwargs):
    return kwargs

create_user(name="홍길동", age=30)
# {'name': '홍길동', 'age': 30}
```

## 핵심 포인트 정리

- Python은 들여쓰기가 코드 구조를 결정합니다. 일관된 들여쓰기(보통 4칸)가 필수입니다
- 리스트 컴프리헨션은 Python의 강력한 기능입니다. `[표현식 for 변수 in 이터러블 if 조건]`
- `get()` 메서드로 딕셔너리 값을 안전하게 접근하면 KeyError를 방지할 수 있습니다
- 가상환경(venv)을 사용하면 프로젝트마다 다른 패키지 버전을 격리할 수 있습니다
