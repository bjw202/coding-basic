# 심화 학습 개요

수업에서 배운 개념을 더 깊이 파고들고 싶을 때 참고하는 심화 학습 가이드 모음입니다. 각 가이드에는 난이도 표시가 있어 현재 수준에 맞는 자료를 선택할 수 있습니다.

| 세션 | 추천 자료 | 난이도 | 이유 |
|------|----------|--------|------|

| Session 2 이후 | [Express 완전정복](/appendix/deep-dive/express-complete) | 기초 | Express로 실전 서버를 구축합니다 |

| Session 7 이후 | [PostgreSQL 기초](/appendix/deep-dive/postgresql-guide), [MySQL 기초](/appendix/deep-dive/mysql-guide) | 중급 | DB 개념 이후 SQL 심화를 다룹니다 |

## 자료 목록

### 기초 (Session 2~3 이후 권장)

- **[Node.js 핵심](/appendix/deep-dive/nodejs-core)** `기초` — Node.js의 동작 원리부터 이벤트 루프, 내장 모듈, npm까지. JavaScript를 서버에서 실행하는 방법을 완전히 이해합니다.
- **[Express 완전정복](/appendix/deep-dive/express-complete)** `기초` — 라우팅, 미들웨어, 에러 처리, 파일 업로드, 실전 CRUD API 구현까지. Express로 실전 웹 서버를 처음부터 끝까지 만들 수 있습니다.
- **[API 호출 실전가이드](/appendix/deep-dive/api-call-guide)** `기초` — fetch, axios, 날씨 API, 공공데이터 API, AI API 연동까지. 다양한 외부 API를 실전에서 호출하는 방법을 다룹니다.

### 중급 (Session 7~8 이후 권장)

- **[MySQL 기초](/appendix/deep-dive/mysql-guide)** `중급` — MySQL 설치부터 CRUD 쿼리, JOIN, 인덱스, 실전 DB 설계까지. 세계에서 가장 많이 사용되는 관계형 DB를 완전히 정복합니다.
- **[PostgreSQL 기초](/appendix/deep-dive/postgresql-guide)** `중급` — PostgreSQL의 고급 기능인 MVCC, JSON 타입, 벡터 검색, Supabase 연동까지. 프로덕션 환경에서 많이 쓰는 DB를 다룹니다.
- **[ORM 실전가이드](/appendix/deep-dive/orm-guide)** `중급` — Raw SQL, 쿼리 빌더, ORM 비교부터 Prisma로 실전 CRUD, 관계 설정, 마이그레이션까지. Node.js에서 DB를 다루는 모든 방법을 정리합니다.

### Python 심화 (기초~고급)

- **[Python 기초](/appendix/deep-dive/python-basics)** `기초` — Python의 기본 문법부터 조건문, 반복문, 리스트, 딕셔너리, 함수까지. Python으로 처음 시작하는 분을 위한 완전정복 가이드입니다.
- **[Python 중급](/appendix/deep-dive/python-intermediate)** `중급` — 에러 처리, 클래스와 객체지향, 데코레이터, 제너레이터, 정규표현식까지. Python을 더 깊이 활용하고 싶은 분을 위한 가이드입니다.
- **[Python 고급](/appendix/deep-dive/python-advanced)** `고급` — asyncio 비동기 프로그래밍, 멀티스레딩, 디자인 패턴, 성능 최적화, 보안까지. 실서비스 운영 수준의 Python을 다룹니다.

## 핵심 포인트 정리

- 심화 학습은 수업 진도를 마친 후 참고하는 자료입니다. 수업 중 모르는 부분이 있을 때 먼저 강사에게 질문하세요
- 난이도 표시를 참고해 현재 수준에 맞는 자료부터 시작하세요
- Node.js와 Express를 깊이 이해하면 이후 모든 백엔드 작업이 훨씬 쉬워집니다
- Python 3부작은 순서대로(기초 → 중급 → 고급) 학습하는 것을 권장합니다
