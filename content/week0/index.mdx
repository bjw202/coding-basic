# Week 0: 본 과정 시작 전 준비

본 과정을 최대한 활용하기 위해 미리 준비해야 할 내용들을 다룹니다. Week 0은 선택이 아닌 필수입니다. 여기서 다루는 개념과 도구들은 Week 1부터 매 세션에서 당연히 알고 있다고 가정하고 진행됩니다.

## 이 섹션에서 배울 것

| 페이지 | 내용 | 핵심 질문 |
|--------|------|-----------|
| **바이브코딩과 AI 도구 입문** | 바이브코딩의 개념, 컴퓨터와 대화하는 방법, AI 효과적으로 활용하기 | "AI에게 어떻게 말해야 내가 원하는 걸 얻을 수 있을까?" |
| **개발환경 준비** | Cursor/VSCode 설치, Warp 터미널 설치, 에디터와 터미널 연동 | "코딩을 시작하려면 무엇을 설치해야 할까?" |
| **터미널과 IT 기초** | 핵심 터미널 명령어, 개발자가 쓰는 IT 용어 사전 | "개발자들이 당연히 안다고 가정하는 기본기가 뭘까?" |

## 준비 완료 체크리스트

Week 1을 시작하기 전에 아래 항목을 모두 확인하세요.

- [ ] 바이브코딩이 무엇인지 한 줄로 설명할 수 있다
- [ ] AI에게 좋은 프롬프트를 작성하는 3가지 원칙을 안다
- [ ] Cursor 또는 VSCode가 내 컴퓨터에 설치되어 있다
- [ ] Warp 터미널(또는 기본 터미널)을 열 수 있다
- [ ] `pwd`, `ls`, `cd`, `mkdir` 명령어를 실행할 수 있다
